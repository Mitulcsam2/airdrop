âš™ How Peer Discovery Works

Each device periodically sends a UDP broadcast packet formatted as:

0xDECAF<username>:<tcp-port>

All devices listening on UDP port 50000 extract:

  • Peer name
  • IP address (from sender socket)
  • Port number

Peers inactive for more than 10 seconds are automatically removed.

On receiving, the device writes it as airdrop_filename.
